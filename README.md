# WikWok

一个简易的仿制 TikTok 的视频播放网站

## 技术栈

前端: Nuxt + Vue3 + Typescript + TailwindCSS
后端: Laravel

## 项目亮点

**响应式布局**: 相应不同屏幕格式，拥有不同的布局
**搜索框样式**: 模仿的抖音官网的搜索框
**vue-advanced-cropper**:一个 vuejs 的图像裁剪组件

## 快速开始

### 后端启动

需要先安装 php 和 composer

```bash
# 选择仓库
cd WikWok
cd WikWok-api
# 安装依赖
npm install
# 启动
npm run build
composer run dev
```

### 前端启动

```bash
# 选择仓库
# 安装依赖
npm install
# 启动前端项目
npm run dev
```
